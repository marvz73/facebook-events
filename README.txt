This plugin will help you fetch events on facebook by location or near to you and enable to filter by distance.

== Description ==

This plugin allow you to fetch events on facebook that near to you current location and enable you to make a filter by distance, coming, past and etc.

== Installation ==

1. Upload `facebook-event-radar file` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[event-radar]` in your posts or pages or you can place on your templates by `<?php do_shortcode('[event-radar]'); ?>`
